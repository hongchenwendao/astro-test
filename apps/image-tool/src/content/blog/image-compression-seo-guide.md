# 为什么图片压缩对 SEO 至关重要？

Google 在 2021 年将 **Core Web Vitals** 正式纳入排名因素。其中最关键的指标之一就是 **LCP（Largest Contentful Paint）**—— 页面最大内容元素的加载时间。

而在大多数网页中，最大的内容元素恰恰就是**图片**。

## 图片大小如何影响排名？

```
未压缩图片 (5MB) → 加载 8 秒 → LCP 差 → 排名下降
压缩后图片 (500KB) → 加载 0.8 秒 → LCP 优 → 排名上升
```

Google 的研究数据显示：

- 页面加载时间从 **1 秒增加到 3 秒**，跳出率上升 **32%**
- 从 **1 秒增加到 5 秒**，跳出率上升 **90%**

## Core Web Vitals 三大指标

| 指标 | 含义 | 与图片的关系 |
|------|------|-------------|
| **LCP** | 最大内容渲染时间 | 直接受图片大小影响 |
| **FID** | 首次输入延迟 | 大图片阻塞主线程 |
| **CLS** | 累积布局偏移 | 未设置尺寸的图片导致偏移 |

## 最佳实践

### 1. 压缩所有图片
使用 [ImageSquish](/tool) 这样的工具，将图片体积减少 70-85%，同时保持视觉质量。

### 2. 使用现代格式
WebP 格式比 JPEG 小 25-34%，比 PNG 小 26%。

### 3. 设置图片尺寸属性
```html
<img src="photo.webp" width="800" height="600" alt="描述" />
```
这能避免 CLS（布局偏移）问题。

### 4. 使用懒加载
```html
<img src="photo.webp" loading="lazy" alt="描述" />
```
首屏以下的图片使用 `loading="lazy"` 延迟加载。
